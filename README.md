# WAVE - Website Assessment Vulnerability Engine

WAVE est un outil en ligne de commande (CLI) visant à faciliter l’analyse de la surface d’attaque de site web.

### Installation

Depuis la racine du projet :

```bash
pip install .
```

En mode développement :

```bash
pip install -e .
```

### Utilisation

Exemple de commande :

```bash
wave scan https://example.com
```

### Objectif

    Scanner une cible (URL/IP)

    Réaliser un inventaire des pages

    Identifier des points d’entrée potentiels pour des vulnérabilités web

    Générer un rapport exploitable (PDF à terme)

### Statut

Projet en phase initiale de développement.
